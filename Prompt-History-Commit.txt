# 인디케이터 디테일 수정-------------------------------------------------------------------------------------------------------------------------
현재 우측 내비게이션 인디케이터(Side Navigation)의 로직과 디자인을 전면 수정해야 해. 다음 가이드라인을 엄격히 지켜줘.

1. 개별 상태 분리 (Active Logic):

현재 모든 인디케이터가 동시에 회전하는 버그가 있어. Intersection Observer나 현재 스크롤 섹션 값을 인덱스별로 정확히 매칭해서, '현재 활성화된 섹션의 인디케이터만' 돌아가게 해줘. 비활성 인디케이터는 고정되어 있어야 해.

2. 나사 회전 모션 (Screw Animation):

Idle 상태: 십자가 모양(+)으로 정지.

Active 상태: 십자가가 x 모양으로 45도 회전하면서, 동시에 살짝 안으로 조여지는(Scale down) 느낌을 줘서 나사가 박히는 듯한 모션을 구현해 줘. (Framer Motion이나 CSS Transition 사용)

3. 정밀 정렬 (Alignment):

position: fixed를 사용하고, flex-direction: column으로 일정한 간격(gap)을 줘서 수직으로 완벽하게 정렬해 줘. 삐뚤빼뚤하지 않게 align-items: center를 확실히 잡아줘.

4. 시각적 디테일:

+ 모양은 아주 얇은 선(1px)으로 표현해서 고딕한 무드를 살려줘.

이 내용을 바탕으로 인디케이터 컴포넌트 코드를 리팩토링해 줘.
